<strong>Sobre</strong>

Aplicativo para buscar repositórios da linguagem kotlin na API do github.

<strong>Tecnologias e ferramentas utilizadas</strong>

* Android Studio
* Picasso
* Retrofit
* RxAndroid
* Kotlin
